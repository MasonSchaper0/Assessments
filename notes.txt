CSS SELECTORS: selectors are patterns used to select the element you want to style.

element: basic form of selector
p {
    font-size: 10px;
}

#example {
    font-size: 10px;
}
/* changes the font-size to 10px */

attribute: select elements with a specified attribute

a[href="www.google.com"] {
    background-color: #df009d;
}
/* this will select any element with an A-tag targeting google */


pseudo: style specific parts of an element or value

p:link {
    color: red;
} (before link is visited, will be red)

p:visited {
    color: blue;
} (after link is visited, will be blue)

variable fonts: Variable fonts are an evolution of the OpenType 
font specification that enables many different variations of a typeface to be incorporated into a single file.

variable font example -->
.text {
    font-variation-settings: 'slnt' 5, 'ital' 5; /* variable font settings allows you to utilize new or custom axes beyond the five registered ones. */
    font-style: oblique 10 10; /*
}

/* the example above with slant the words by 5 degrees and make it italic. It also changes the angle of the letters with oblique. */

text clipping: sets how hidden overflow content is signaled to users. It can be clipped.

2D rotation: add a perspective attribute then rotate it.

complex 2D: transform: skew the text, then make an additional box to rotate it.

basic animation: you can make key frames to set an attribute from each percentage of the motion or make your animation go from 1 part to another.

    //example 
    @keyframes spin {
        from { 
            transform: rotateY(-260deg);
        }

        to {
            transform rotateY(500deg);
        } 
    }
}

/: example above will start a Y angle of -260 degrees and move to 500 degrees.
